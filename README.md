# AWS S3 Bucket Experiment

## Experiment Title

Creating Two S3 Buckets and Uploading Objects using AWS Web Console

---

## Aim

To create two Amazon S3 buckets and upload files into them using AWS Management Console.

---

## Requirements

* AWS Account
* Internet Connection
* Access to AWS Management Console

---

## Procedure

### Step 1: Login to AWS Console

* Go to https://aws.amazon.com
* Sign in to AWS Console

### Step 2: Open S3 Service

* Search for **S3**
* Open Amazon S3

### Step 3: Create First Bucket

* Click **Create Bucket**
* Enter unique name
* Select region
* Create bucket

### Step 4: Create Second Bucket

* Repeat same steps for second bucket

### Step 5: Upload File to First Bucket

* Open first bucket
* Click Upload
* Add file
* Upload

### Step 6: Upload File to Second Bucket

* Open second bucket
* Upload another file

---

## Result

Two S3 buckets were created and files were uploaded successfully.

---

## Conclusion

This experiment demonstrates how to use Amazon S3 for cloud storage, including creating buckets and uploading objects efficiently.
